Product Sets for POS (multisession extension)

Synchronize product sets across several POSes

The module allows to synchronize product sets across several POSes. For more information about product sets see Product Sets for POS.

How It Works

Open Point of Sale → Configuration → Multi-sessions and select product sets in the Available Product Sets field


Open synced POS sessions and see that they have the same products sets
