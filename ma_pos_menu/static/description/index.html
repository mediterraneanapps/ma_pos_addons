Product Sets for POS

Add own set of products per each POS

In out-of-box Odoo you can mark a product as available in POS without ability to select specific POSes. The module allows to create product sets and associate those to POSes where it should be available for. It means that you can setup the displaying of certain products on certain POSes.

How It Works

Open Point of Sale → Configuration → Product Sets and create any number of product sets



You can also select the created product sets at POS configuration form directly


Open POS session and see that it displays only the product sets selected
