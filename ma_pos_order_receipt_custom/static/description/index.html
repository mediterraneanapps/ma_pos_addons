Customizable POS Kitchen Ticket

Customize POS kitchen ticket as you need

Key features:
  • All features from POS Receipt Custom module
  • Ability to customize pos kitchen ticket view

How It Works

Go to Point of Sale → Configuration → Order Printers.

  • Open order printer form
  • Check Custom Order Receipt box
  • Select a custom template or add new one in Print Template field
